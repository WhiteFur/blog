2017-06-26
#gama #egaming #job

職務說明

  1. 撰寫博奕網站前端頁面
  2. 撰寫賭場系統包含帳務管理、會員管理與彩金管理
  3. 撰寫網站後端(Web Service)程式的開發、佈建、除錯及資料庫(MySQL)等

職務要求

  1. 專長於Javscript, Nodejs 開發
  2. 熟悉其中一項Web Framework(Angular, Vue, React)，並擁有一年以上的實際開發經驗
  3. 具關聯式資料庫系統應用開發者尤佳
  4. 曾使用Selenium做Web前端測試經驗尤佳
  5. 具備撰寫Unit Test經驗(e.g, 使用mocha做API unit test)
  6. 有UI或UX經驗與技能，前後端開發經驗皆有者也佳
  7. 曾使用Jenkins, Docker, Ansible等工具建置CI/CD流程經驗尤佳

使用工具

  1. Angular, Express
  2. MySQL
