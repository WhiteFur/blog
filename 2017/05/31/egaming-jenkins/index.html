egaming-jenkins

2017-05-31
#jenkins #egaming

連上jenkins website

  • http://localhost:8080
  • user: whitefurlin
  • password: jumbo.net

compile 前台程式

  • click EGamingController

  • 馬上建置

  • 建置中

  • 待建置完畢後，按下「回到儀表板」

compile 後台程式

  • click EGamingHostController

  • 馬上建置

打包安裝檔

  • 待EGamingController和EGamingHostController建置完成後，記得它們的最新Number，等等會使用到

  • 點擊InstallPackage標題右側的倒三角

  • Build with Parameter

  • 使用剛剛記下的Number
  • PlayerBuildNumber: EGamingController的Number
  • HostBuildNumber: InstallPackage的Number

  • 點擊「建置」

  • 待建置完成後，點擊「最新成品」即可看到建置完的成品

  • 實際位置在C:\Program Files (x86)\Jenkins\jobs\InstallPackage\builds下的最大號碼的資料夾裡

  • 例如本例為C:\Program Files (x86)\Jenkins\jobs\InstallPackage\builds\38\archive
