electron-node-version

2017-05-03
#electron

執行electron時

跳出NODE_MODUEL_VERSION不合的error

這個error的意思是說:
DTraceProviderBindings.node(bunyan用到的dtrace-provider) 是用NODE_MODULE_VERSION 51來compile的
但是目前electron所運行的版本是NODE_MODULE_VERSION 53

正常情況下應該是要用electron-rebuild來解決

如果是fish shell的話，command substitute的用法與bash不同
去掉dollar sign，而且要用eval來執行

但是根據Nodejs Release Note
目前最新的NODE_MODULE_VERSION也才51，哪來的53 (此時最新版本是Nodejs v7.9)
github上的網友也是這樣說的

最後找到解法
在electron project建一個.npmrc

然後再重新npm install就可以正常運作了
